## 수집 원칙
- 공개적으로 접근 가능한 데이터 우선
- API가 있으면 API 우선
- robots / 약관 / 접근 정책 확인
- 개인 대화 / 비공개 영역은 수집 대상에서 제외
- 중복 사례는 후처리에서 제거

## 메모
- 실제 수집 대상은 접근 가능성과 품질에 따라 바뀔 수 있다.
- 거래 플랫폼 데이터는 구조가 자주 변하므로 adapter 구조를 염두에 둔다.
